server/Api

imgur.Api.DEFAULT_OPTIONS

Default options used for imgur api http calls

new imgur.Api(apiKey, [options])

  • apiKey String - your personal imgur api key. * options Object - a custom options object that will be merged with the default options imgur.Api.DEFAULT_OPTIONS, Optional

api.delete(hash)

  • hash String - the id/hash of the image that should be deleted

api.getImageInformation(hash, callback, [ctx = null])

  • hash String - the id/hash of the image you want to get the information for * callback Function - callback function
  • ctx Object - the context of the callback function, Optional
